# Manawork
a Simple Kanban Board written using C# / Asp.net Core 5

# Usage
- install dotnet sdk
- Clone 'https://github.com/aminasadiam/Manawork'
- Go to Src
- Edit ConnectionString in 'appsetting.json'
- Create Database : `dotnet ef database update`
- Run App : `dotnet run`
